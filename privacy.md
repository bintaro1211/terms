# プライバシーポリシー

本プライバシーポリシーは、本アプリ（以下「本アプリ」）が取得・利用する情報、
および利用者のプライバシー保護に関する方針を定めるものです。

## 1. 取得する情報

本アプリは、以下の情報を取得する場合があります：

- TikTokユーザーID（open_id）
- 認証用トークンおよびリフレッシュトークン
- 投稿する動画ファイルおよびキャプション
- TikTok API から提供される公開情報

## 2. 情報の利用目的

本アプリが取得した情報は、以下の目的のために使用します：

- TikTok API を利用した動画投稿・管理機能の提供
- 投稿スケジュールや自動投稿機能の提供
- 機能改善およびエラー対策

利用者の同意なく、本アプリはこれらの情報を他の目的に使用しません。

## 3. 情報の保管・共有・第三者提供

本アプリは、取得した情報を第三者に販売・共有・提供しません。
ただし、以下の場合を除きます：

- 利用者の明確な同意がある場合
- 法令に基づき情報提供が求められた場合

## 4. Cookie 等の利用

本アプリは、単独ではCookieの利用を行いませんが、
TikTok API による認証画面では TikTok の Cookie が使用される場合があります。
詳細は TikTok のプライバシーポリシーをご確認ください。

## 5. 利用者によるデータの削除

利用者は、アプリ内で使用されたデータの削除を要求できます。
削除を希望される場合は、本アプリ提供者までお問い合わせください。

## 6. 外部サービス・リンク

本アプリには TikTok 等の外部サービスへのリンクが含まれる場合があります。
それら外部サービスのプライバシー保護については、本アプリは責任を負いません。

## 7. 変更

本プライバシーポリシーの内容は予告なく変更されることがあります。
変更後のプライバシーポリシーは、本ページに掲載された時点で効力を持つものとします。
